# AI Resume Analyzer 🤖📄

A stylish AI-powered Resume Analyzer built using HTML, CSS, JavaScript, and Node.js.

# Features
- Upload resume
- Analyze resume content
- Detect skills
- Suggest improvements
- Modern black & white UI
- Beginner-friendly project

# Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

# Project Preview
Users can upload resumes and receive smart suggestions instantly.
